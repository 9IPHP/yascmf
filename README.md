# 芽丝内容管理框架(YASCMF)


> 芽丝内容管理框架( 英文简称 `YASCMF` )， 基于 `Laravel 4` 开发而成，它比较适合拿来做一些小众项目开发。目前框架实现了一个简单的内容管理系统（ `CMS` ），支持多种内容模型，文章、单页、分类、碎片与标签，您现在完全可以拿 `yascmf` 来完成一个简单的博客网站。

> YASCMF将于年后正式上线发布时，目前官方给出一个由其驱动的[博客演示网站](http://cmf.douyasi.com)，欢迎访问了解。
